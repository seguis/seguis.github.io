// sobre mí

Un perfil técnico completo

Desarrollador en formación DAM con experiencia real en redes, sistemas y desarrollo web full-stack.

// Formación

Técnico en Desarrollo de Aplicaciones Multiplataforma (DAM). Apasionado por aprender continuamente — gran parte de mi stack lo he construido con ayuda de IA, experimentación y proyectos reales.

// Filosofía

Prefiero Linux sobre Windows. Me gusta controlar la infraestructura al completo: desde el router hasta la aplicación web. Si algo se puede automatizar, se automatiza.

// Infraestructura propia

Tengo mi propio NAS con SnapRAID + MergerFS, servidor Proxmox con VMs y contenedores LXC, y firewall OPNsense. Aprendo haciendo.

// Próximos pasos

Desarrollo de una plataforma RADIUS propia para ofrecer autenticación como servicio. También ampliando conocimientos en DevOps y cloud.
